## 2. Sammendrag

Hugo Pereira bruker TFS/Norsk Toppfotballsenter sin videoanalyse fra VM 2014 (Kompetansekrav i Rolle) for å oversette vinger-statistikk til praktiske treningsøvelser. Rapporten argumenterer for **implicit læring** gjennom constraints-baserte småspill fremfor eksplisitt instruksjon. Målet er at spilleren lærer gjennom spilldesign (baneformat, regler, touch-begrensning) heller enn «go here, do that»-instrukser. Rosenborg-konteksten (brede vinger, 1v1, tidlige innlegg) gjør dette særlig relevant.

## 4. Prinsipper og konsepter

### 4.1 Implicit læring vs. eksplisitt instruksjon
- Constraints (regler, baneformat, touch-begrensning) erstatter verbal instruksjon
- Spilleren «oppdager» løsninger gjennom spillets struktur
- Lengre læringsprosess, men mer robust under press

### 4.2 Trefase-modellen (TFS)
For hver hovedhandling (assist, innlegg, scoring):

| Fase | Beskrivelse | Nøkkelspørsmål |
|------|-------------|----------------|
| **Fase 1** | Bevegelse før ball | Hvor er motstander? Hvor går vingen? |
| **Fase 2** | Fra mulighet til realitet | Press, handlingstype, tid med ball, touch |
| **Fase 3** | Avgjørende handling | Press, type pasning/skudd, område |

### 4.3 Propensity Principle
Spilldesign skal **øke sannsynligheten** for at ønskede handlinger skjer naturlig, ikke tvinges frem.

## 5. Øvelser for treningsdelen (UEFA-knapp)

### Øvelse 1: Vinger-assist – Mellomrom

| Felt | Beskrivelse |
|------|-------------|
| **Kode** | UEFA-A10-01 |
| **Kategori** | Angrep |
| **Spillere** | 14-16 (Gk+7v7+Gk) |
| **Bane** | 45 x 35 m |
| **Format** | Gk+7v7+Gk |
| **Formasjon** | 1-4-3 eller 1-4-1-2 |
| **Utstyr** | 2 stk 11er-mål, kjegler for soneinndeling |
| **Varighet** | 15-20 min |
| **Intensitet** | Medium-høy |

**Organisering:**
- Bane med min. 10m bakrom tilgjengelig
- Forsvarslinjen må holde seg nær eget mål (organisert forsvar)

**Regler/Constraints:**
1. **Dutch Rule:** Spill starter alltid fra keeper
2. 3 spillere må være låst i egen halvdel når laget har ball (hindrer kontra)
3. Forsvar kan bare bruke 2 korridorer (tvinger vinger til mellomrom)

**Poeng:**
- Vanlig mål = 1 poeng

**Coaching points:**
- Se etter: Vingers bevegelse inn i mellomrom når back snur
- Nøkkelord: «Timing», «Rettvendt før touch»

**Progresjon:**
- Enklere: Ingen touch-begrensning
- Vanskeligere: 1 touch i forsvarende halvdel, fritt i angrep

### Øvelse 2: Vinger-assist – Rask beslutning

| Felt | Beskrivelse |
|------|-------------|
| **Kode** | UEFA-A10-02 |
| **Kategori** | Angrep |
| **Spillere** | 10-12 (Gk+5v5+Gk) |
| **Bane** | 35 x 25 m |
| **Format** | Gk+5v5+Gk |
| **Formasjon** | Fritt |
| **Utstyr** | 2 stk 11er-mål |
| **Varighet** | 12-15 min |
| **Intensitet** | Høy |

**Organisering:**
- Kompakt bane = høy intensitet og tett marking

**Regler/Constraints:**
1. Forsvar kan bare bruke 2 korridorer (posisjonsforsvar)
2. Alle må bruke begge bein ved >1 touch
3. Touch-begrensning: 1 eller 3 touch (variabel)

**Poeng:**
- Vanlig mål = 1 poeng

**Coaching points:**
- Se etter: Orientering før mottak, rask førstekontakt
- Nøkkelord: «Opp med hodet», «Løsning klar før ball»

---

### Øvelse 3: Innlegg – Vingerkorridorer

| Felt | Beskrivelse |
|------|-------------|
| **Kode** | UEFA-A10-03 |
| **Kategori** | Angrep |
| **Spillere** | 12-14 (Gk+6v6+Gk) |
| **Bane** | 40 x 40 m (bruk 16m-boks som referanse) |
| **Format** | Gk+6v6+Gk |
| **Formasjon** | 1-3-3 eller 1-2-3-1 |
| **Utstyr** | 2 stk 11er-mål, kjegler for sidekorridorer |
| **Varighet** | 15-20 min |
| **Intensitet** | Medium-høy |

**Organisering:**
- Markerte sidekorridorer (5-7m bredde)
- Forsvar faller mot eget mål

**Regler/Constraints:**
1. **Dutch Rule (tilpasset):** Spill starter fra treneren på sidelinja
2. Markerte sidekorridorer
3. Forsvar kan bare operere i 2 korridorer
4. Touch-begrensning i sidekorridor: 1 touch

**Poeng:**
- Vanlig mål = 1 poeng
- Mål etter baklengs innlegg (cutback) = 2 poeng

**Coaching points:**
- Se etter: Timing på innløp, variasjon tidlig/sent innlegg
- Nøkkelord: «Les rommet bak back», «Tidlig innlegg»

| Felt | Beskrivelse |
|------|-------------|
| **Kode** | UEFA-A10-04 |
| **Kategori** | Angrep |
| **Spillere** | 6-8 + 2 jokere (Gk+2v2+Gk + 2 Jk) |
| **Bane** | 20 x 45 m (bruk 16m-boks) |
| **Format** | Gk+2v2+Gk + 2 jokere på sidene |
| **Formasjon** | – |
| **Utstyr** | 2 stk 11er-mål, markerte sidekorridorer |
| **Varighet** | 10-12 min |
| **Intensitet** | Høy |

**Organisering:**
- Vinger som jokere langs sidelinjene
- Smalt spillfelt = høy frekvens av innleggssituasjoner

**Regler/Constraints:**
1. **Dutch Rule:** Starter fra keeper
2. Jokere (vinger) maks 1-3 touch
3. Mål etter førstekontakt = 2 poeng

**Coaching points:**
- Se etter: Jokernes orientering før de mottar
- Nøkkelord: «Les boksen før innlegg»

### Øvelse 5: Vinger-scoring – Finne scoringsrom

| Felt | Beskrivelse |
|------|-------------|
| **Kode** | UEFA-A10-05 |
| **Kategori** | Angrep |
| **Spillere** | 12-14 (Gk+6v5+Gk) |
| **Bane** | 30 x 40 m (bruk 16m-boks) |
| **Format** | Gk+6v5+Gk (undertall for vingerens lag) |
| **Formasjon** | – |
| **Utstyr** | 2 stk 11er-mål |
| **Varighet** | 12-15 min |
| **Intensitet** | Høy |

**Organisering:**
- Vingerens lag i undertall (5 utespillere)
- Motstanderlag med overtall i forsvarende halvdel

**Regler/Constraints:**
1. **Dutch Rule**
2. Soneoverskudd: Forsvarende lag har +1 spiller i egen halvdel

**Poeng:**
- Vanlig mål = 1 poeng

**Coaching points:**
- Se etter: Vinger finner rom mellom/bak linjer under press
- Nøkkelord: «Reposisjoner», «Kamuflér inngang»

**Progresjon:**
- Vanskeligere: Forsvar i 4-linje (realistisk pressing)

### 6.2 Motstander-analyse
- Hvor høyt står motstanderens back?
- Svak fot på back? (angrip den siden)
- Avstand mellom back og 6-er? (mellomrom å utnytte)
- Hvordan forsvarer de innlegg? (sone/mann)

### 6.3 Rollebeskrivelse (Roles-komponenten)
**Vinger – offensive oppgaver:**
- Hold bredde til 20m fra mål, skjær inn ved lavblokk
- Trigger-løp når 6-er vender med ball
- Prioriter: bakrom → mellomrom → framrom
- Mentale cues: «attack space, attack man, find box»

**Vinger – defensive oppgaver:**
- Tilbakesporing ved balltap (5-6 sek press)
- Dødball: sonemarkering + kontrarolle (startposisjon, første løp)

## 8. Coaching Tips

- **Cue (mottak):** «Opp med hodet før ballen kommer»
- **Cue (innlegg):** «Les boksen – første stolpe, bakre, retur»
- **Cue (1v1):** «Første touch framover – angrip rom»
- **Cue (scoring):** «Lavt og hardt i hjørnet»
- **Ikke:** Overinstruer under spill – la constraints gjøre jobben
