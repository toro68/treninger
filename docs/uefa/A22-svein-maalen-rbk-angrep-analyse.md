## 2. Sammendrag

- RBK skiftet hovedtrener fem ganger fra 2006-2011; spillestil og angrepsprofil varierer tydelig mellom regimene.
- Poengfangst henger sterkest sammen med differanse i sjanser/mål for og mot; Eggen og Hamrén kombinerte høy sjansedifferanse med strammere defensiv balanse.
- Tørum og Jönsson skapte mange sjanser og dødballmål, men slapp også til flest sjanser mot etablert forsvar.
- Henriksen-perioden var kort men ekstremt effektiv (33% store sjanser), mens Hamrén prioriterte trygg posession, lav risiko og lavt antall kontringer begge veier.
- Jönssons 2011-lag leverte høyeste måltall offensivt (2,3 mål/9,7 sjanser pr. kamp), men lekkasjer i etablert forsvar ga bare 1,6 poeng i snitt.

## 4. Prinsipper og konsepter

### 4.1 Hovedprinsipper
- Prioriter sjansedifferanse: mål om minst dobbelt så mange sjanser som motstander.
- Juster gjennombruddshissighet etter ønsket risikoprofil; flere kontringer gir større sjanser men eksponerer balanse.
- Balansert dødballprogram: dødballer var nøkkelforskjell hos Tørum og Jönsson, men må forankres i spillertyper.
- «Def i off» (Hamrén): samle laget rundt ball, sikre umiddelbar gjenvinning og lav kontringsrisiko.
- Bruk formasjonsskiftet bevisst (4-3-3 vs. 4-2-3-1) for å styre mellomromsbeskyttelse og antall spillere i siste tredel.

### 4.2 Fasemodell
- **Fase 1:** Oppbygging med trygg pasningsstruktur, keeper og stoppere etablerer balanse før progresjon.
- **Fase 2:** Gjennombrudd via indreløper/spiss-relasjoner; styr valg mellom direkte kontring eller etablert spill.
- **Fase 3:** Avslutningssoner med klar rollefordeling for vinger/backer og aggressiv boks-tilstedeværelse.

## 5. Øvelser for treningsdelen (UEFA-knapp)

### Øvelse 1: Differansejakt 4v4+3

| Felt | Beskrivelse |
|------|-------------|
| **Kode** | UEFA-angrep-01 |
| **Kategori** | Angrep |
| **Spillere** | 11 (4v4 i kvadrat + 3 nøytrale) |
| **Bane** | 35x30 m delt i to halvdeler |
| **Format** | Rondo/possession med scoring på gjennombruddspasning |
| **Formasjon** | 4-3-3 prinsipper |
| **Utstyr** | Kjegler, vester, minitreningsmål |
| **Varighet** | 3x6 min |
| **Intensitet** | Medium |

**Organisering:**
- To lag á fire spillere i yttersonene; tre nøytrale i midten skaper overtall for ballførende lag. Scoring ved å spille gjennom linjen til medspiller som avslutter i miniatyrmål.

**Regler/Constraints:**
1. Maks tre touch for nøytrale, to touch for lag i medgang.
2. Ballen må innom minst én indreløper (nøytral) før gjennombrudd.
3. Ved brudd må laget som misser ballen falle på plass i 3 sekunder før press.

**Poeng/Mål:**
- Gjennombruddspasning som treffer spiss = 1 poeng
- Vinne ball innen tre sekunder = bonuspoeng

**Coaching points:**
- Korte støttetrekanter, scanning før mottak.
- Synkroniser høye bevegelser (vinger/spiss) med midtbanens pasningsrytm e.

**Progresjon:**
- Enklere: Fri touch, færre nøytrale.
- Vanskeligere: Legg til restforsvar (to stoppere) som kan bryte linjepasning.

### Øvelse 2: Kontring til balanse 6v4

| Felt | Beskrivelse |
|------|-------------|
| **Kode** | UEFA-angrep-02 |
| **Kategori** | Overgang |
| **Spillere** | 10 + keepere |
| **Bane** | 50x40 m delt i tre soner |
| **Format** | Gk+5v4+Gk, med retning begge veier |
| **Formasjon** | 4-3-3 vs. 4-4-2 |
| **Utstyr** | Mål, kjegler, tidsklokke |
| **Varighet** | 4x5 min |
| **Intensitet** | Høy |

**Organisering:**
- Angripende lag starter med keeper; mål er å finne kontringsrom på 6 sek. Ved avslutning må laget umiddelbart falle og sikre «def i off» soner.

**Regler/Constraints:**
1. Maks tre pasninger før avslutning i kontringsfase.
2. Ved balltap må nærmeste tre spillere presse, resten faller.
3. Defensivt lag får to ekstra poeng hvis de klarer å etablere 6 spillere bak ball innen 4 sekunder.

**Poeng/Mål:**
- Kontringsmål innen 8 sekunder = 2 poeng
- Mål etter etablert spill = 1 poeng

**Coaching points:**
- Første pasning framover eller diagonalt.
- Kommunikasjon mellom sentral midtbane og stopper om hvem som blir igjen.

**Progresjon:**
- Enklere: Fjern tidspress.
- Vanskeligere: Legg til nøytral midtbane for støtte til ballfører.

### Øvelse 3: Dødballpakke RBK

| Felt | Beskrivelse |
|------|-------------|
| **Kode** | UEFA-angrep-03 |
| **Kategori** | Dødball |
| **Spillere** | 14-16 |
| **Bane** | 35x30 m område rundt 16-meter |
| **Format** | Cornersekvenser med restforsvar |
| **Formasjon** | 4-3-3, 6 spillere i boks |
| **Utstyr** | Kornere flagg, mannequins |
| **Varighet** | 3 blokker á 8 repetisjoner |
| **Intensitet** | Medium |

**Organisering:**
- To varianter: a) nærstol flick (Tørum/Jönsson signatur), b) motbakre soner med screening.

**Regler/Constraints:**
1. Serven må slå samme trigger (armtegn) før hver repetisjon.
2. Restforsvar (2 stoppere + sentral midt) må holde 2. linje.
3. Nytt forsøk hvis markering ikke bryter av i sprint.

**Poeng/Mål:**
- Scoring = 2 poeng
- Klarering med kontrollert kontring = 1 poeng for forsvar

**Coaching points:**
- Timing mellom server og løpere, blokk vs. skjerm.
- Umiddelbar re-press hvis avslutning avverges.

**Progresjon:**
- Enklere: Uten motstandere i feltet.
- Vanskeligere: Legg inn sonemarkering + kontra.

### Øvelse 4: Gjennombruddstriangel 8v6

| Felt | Beskrivelse |
|------|-------------|
| **Kode** | UEFA-angrep-04 |
| **Kategori** | Angrep |
| **Spillere** | 14 + keepere |
| **Bane** | 60x45 m delt i tre korridorer |
| **Format** | 8v6 posisjonsangrep med flyttbare jokere |
| **Formasjon** | 4-2-3-1 |
| **Utstyr** | Mål, kjegler, mannequins |
| **Varighet** | 3x8 min |
| **Intensitet** | Medium/Høy |

**Organisering:**
- Angripende lag bygger opp med to stoppere, to sentrale midt, tre offensive (vinger + 10-er) og spiss. Forsvarende lag i 4-4-2. Jokere (backer) kan kun operere i ytre korridor.

**Regler/Constraints:**
1. Back + ving + indreløper må involveres før innlegg/skudd.
2. Angripende lag får kun tre trekk i mellomromssonen før de må slå gjennom.
3. Ved balltap må nærmeste fire spillere presse umiddelbart eller øvelsen stoppes.

**Poeng/Mål:**
- Mål etter gjennombrudd via korridor = 2 poeng
- Mål etter sentral kombinasjon = 1 poeng

**Coaching points:**
- Timing på tredjemannsbevegelse bak siste linje.
- Sentral midtbane må skanne restforsvar før de frigjør backene.

**Progresjon:**
- Enklere: Fjern pressregel i mellomrom.
- Vanskeligere: Reduser touch til to i siste tredel.

---

### Øvelse 5: Press-reset 7v7+keeper

| Felt | Beskrivelse |
|------|-------------|
| **Kode** | UEFA-angrep-05 |
| **Kategori** | Overgang/Forsvar i angrep |
| **Spillere** | 16 |
| **Bane** | 55x40 m | 
| **Format** | 7v7 med keeper, to restforsvars-soner |
| **Formasjon** | 4-3-3 speilet |
| **Utstyr** | Mål, kjegler, tidsmarkør |
| **Varighet** | 4x5 min |
| **Intensitet** | Høy |

**Organisering:**
- Spill kontinuerlig kampferdige situasjoner. Ved hver avslutning instrueres angrepslaget til å «reset» i 5-0-2 restforsvar før motstander kan sette i gang ny kontring.

**Regler/Constraints:**
1. Etter avslutning har laget tre sekunder til å fylle restforsvarssonene.
2. Oppnår de ikke posisjonering i tide, får motstander starte kontring med overtall (legg til joker).
3. Pressfelle trigges kun når spiss signaliserer; ellers må laget falle til blokk.

**Poeng/Mål:**
- Mål kombinert med korrekt restforsvar = 2 poeng
- Vellykket pressfelle som fører til gjenvinning innen fem sekunder = 1 poeng

**Coaching points:**
- Kommunikasjon mellom stopper og sentral midt for fordeling av restrom.
- Stramme avstander mellom linjer før pressfelle slås på.

**Progresjon:**
- Enklere: Øk restforsvarstiden til fem sekunder.
- Vanskeligere: Tilfør ekstra motstander-joker i kontringer for å stresse struktur.

## 6. CornerOrganization / Dødball

### 6.1 Oppsett
- Forsvar: kombiner sone (1+3) med mann for å håndtere fysisk sterke motstandere fra Tørum/Jönsson-perioden; stoppere blir i rest, vingene på stolper.
- Offensivt: varianter med nærstol flick og bakre skjerming (Sapara/Iversen-rolle), alternativt korte hjørner for å utnytte kombinasjoner fra Eggen-perioden.
- Keeper: start én meter foran målstreken, gi klare signaler for første ball og droppe, koordinere med restforsvar før avspark.

### 6.2 KPI-er for dødball
| Situasjon | Mål | Kilde |
|-----------|-----|-------|
| Offensiv corneruttelling | ≥0,4 store sjanser per kamp | Oppgave, dødballstatistikk Tørum/Jönsson |
| Defensiv dødball mot | ≤1,2 sjanser mot pr. kamp | Oppgave, «Målsjanser mot RBK» |

### 6.3 Øvelser/sekvenser
- Kjør blokker á 8 cornerrepetisjoner med fokus på Sapara-lignende serve og screening på keeper.
- Simuler restforsvarstriggere: stopper + sentral midt må være posisjonert før corner tas, ellers nuller repetisjonen.

### 7.2 Motstander-analyse
- Registrer hvor mange kontringer motstanderen aksepterer; Eggen/Henriksen krevde ekstra beskyttelse i bakrom.
- Se etter presshøyde og hvor lett de slipper ballfører å vende (styrer hvorvidt RBK skal spille gjennom eller rundt).
- Trigger: Hvis motstander bruker mann-orientering på dødball, bytt til kort variant.

### 7.3 Rollebeskrivelse
- **Sentral midtbane:** dirigere tempo (play-and-move) og sikre restforsvar.
- **Vinger:** konsekvent løpsvilje bak linjen når ubalanse oppstår; bidra med returløp når vi mister ballen.
- **Spiss:** fall/flate bevegelser for å åpne mellomrom; frontpress første 3 sekunder ved balltap.
