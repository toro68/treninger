## 2. Sammendrag

Rushfeldt kombinerer 20 års erfaring som toppspiss med videodata fra egne mål og scoringer fra Barcelona, Real Madrid, Man City/United (2011/12). Han destillerer 14 punkter som kjennetegner gode målscorere: posisjonering, orientering, avslutningsteknikk, mentalitet og gjentagende trening. Analysen viser at målscoring er trenbar; toppavsluttere har repeterbare rutiner og jobber kontinuerlig med touch, tempo, bevegelse og etterarbeid. Oppgaven oversettes til KPI-er, øvelser og kampplanpunkter som gjør spissrollen mer strukturert i norsk fotball.

## 4. Prinsipper og konsepter

### 4.1 Hovedpunkter (fra foredraget)
1. Orientering før mottak.
2. Tåle å mislykkes – mental robusthet.
3. Første touch inn i avslutning.
4. Varier bevegelse (bakrom, forrige stolpe, motsatt).
5. Etterarbeid – jakt returer.

### 4.2 Læringsteori
- Rushfeldt knytter forbedring til kognitiv/sosial læring; målscoring er trenbart via bevisst repetisjon, video, feedback og selvregulering.

## 5. Øvelser for treningsdelen

### Øvelse 1: Rush-korridor
| Felt | Beskrivelse |
|------|-------------|
| **Kode** | UEFA-A07-01 |
| **Kategori** | Station |
| **Spillere** | 8–12 |
| **Bane** | 30 x 20 m (to korridorer) |
| **Format** | Sekvens m/ server + spiss |
| **Formasjon** | 2 servere, 1 spiss |
| **Utstyr** | Kjegler, mål |
| **Varighet** | 12 min |
| **Intensitet** | Høy |

**Organisering:**
- To korridorer: A server inspill, spiss må orientere, 1–2 touch avslutning.
- Fokus på varier bevegelse + returløp.

**Regler:**
1. Spiss må bytte korridor etter hver avslutning.
2. Bonus for scoring på første touch + returmål.

**Coaching points:**
- Første touch mot mål.
- Ferdigstilling lavt/hardt.
- Etterarbeid.

### Øvelse 2: 3x3 målsoner
| Felt | Beskrivelse |
|------|-------------|
| **Kode** | UEFA-A07-02 |
| **Kategori** | Game |
| **Spillere** | 12–14 |
| **Bane** | 40 x 30 m oppdelt i tre soner |
| **Format** | Gk+5v5+Gk |
| **Formasjon** | 2-2-1 |
| **Utstyr** | Mål, kjegler |
| **Varighet** | 4 x 4 min |
| **Intensitet** | Høy |

**Organisering:**
- Spiss må være innenfor 16-meteren før scoring teller.
- Kant må fylle motsatt stolpe.

**Regler:**
1. Ball må innom mellomromsone før avlutning.
2. Spiss/kant bytter rolle etter hvert mål.

**Coaching points:**
- Bevegelse før ball.
- Orientering + kroppsstilling.
- Avslutning 1–2 touch.

## 8. Evaluering

Hold evalueringen enkel og kildetro:
- Ser vi flere avslutninger på 1–2 touch i boks?
- Er spiss/kant konsekvent på returer (etterarbeid)?
- Kommer spillere oftere i avslutningsposisjon innenfor 16-meteren?

## 9. Coaching Tips / Trenerveiledning
- **Cue:** «Se før du får den»
- **Cue:** «Første touch inn – skyt!»
- **Cue:** «Miss? Finn ballen igjen.»
- **Ikke:** La spissen stå passivt.
