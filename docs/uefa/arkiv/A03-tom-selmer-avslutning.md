# Avslutningsferdighet – Tom Selmer (UEFA A 2012)

Kilde: «Sekundene som avgjør fotballkamper». Notatet bruker kvantitative funn (Tippeliga vs Champions League) og anbefalinger for avslutningsferdighet.

## 1. Viktige funn
- CL-lag scorer oftere i sone 1cv/1ch (mellom femmeter og straffemerke), med kontroll og plassering, mens TL scorer mer på headinger.
- Utløsende forskjell: avslutningsferdighet (individuell dimensjon) og kvalitet på forspillet (innlegg, gjennombrudd).
- Konklusjon: tren på å utnytte lite bakrom (direkte gjennombrudd) og levere innlegg med høy kvalitet.

## 2. Kampplan for avslutninger
- MatchPrep: legg inn mål for hvor vi skal avslutte (f.eks. 6 avslutninger fra sone 1cv/ch). Planlegg kombinasjoner for å komme til disse rommene (vegger, cut-backs).
- TeamOrganization: beskriv mønster for å komme bak back (overlapping, tredjemannsløp). Spesifiser roller (hvem angriper første stolpe, hvem dropper).

## 3. Trening/MD-1
- Bruk constraints (maks 2 touch, press i ryggen) for å trene «sekundene som avgjør». Oppgi i CoachingTips.
- Egen analyse: logg alle avslutninger fra kamp (posisjon, type, resultat). PostMatch: sammenlign med CL-funn (er vi i rett rom?).

## 4. Egentrening/utvikling
- Selmer beskriver oppfølging av to spillere via egentrening. Kampseksjonen kan pålegge spisser å registrere eget ekstraarbeid og koble til kamp-KPI.

## 5. Mentalt
- Automatiser handling (kort tid). CoachingTips: «ro i boksen» – minne om å se keeper, plassér. Bruk video fra CL for referanse.

## 6. Implementering
- MatchPrep: «Avslutningsfokus» (rom, type, spillere). 
- Roles: utdyp oppgaver for spiss/ving (søk sone 1cv, plassering lavt hjørne) med referanse til CL.
- CoachingTips: cues («bli lenge på beinet», «curl lavt»).
- PostMatch: logg og evaluer. Skriv tiltak (f.eks. mer cut-back trening).

Dette notatet gjør Selmers avslutningsanalyse om til konkrete kampgrep for å øke effektiviteten.
