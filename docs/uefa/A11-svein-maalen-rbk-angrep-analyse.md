## 2. Kort oppsummert (hva analysen faktisk viser)

Rosenborg skiftet hovedtrener fem ganger (Tørum, Henriksen, Hamrén, Eggen, Jönsson), og analysen bruker **målsjanser for/mot** som «speil» på spillestil og konsekvensene av den.

Kjernen i materialet er at **sjanse-differanse (for–mot)** henger tett sammen med poengfangst: det holder ikke å skape mye hvis man samtidig slipper til mye.

## 3. Nøkkeltall du kan styre etter

### 3.1 Poengfangst

- Best: **Eggen 2,4 poeng pr. kamp**, **Hamrén 2,1**.

### 3.2 Sjanser for (pr. kamp) + effektivitet

- Sjanser for: **Tørum 8,4 / Henriksen 6,1 / Hamrén 8,1 / Eggen 9,5 / Jönsson 9,7**
- Sjanseuttelling: **25,26% / 27,87% / 22,96% / 22,37% / 23,79%**
- Sammenheng: **flere sjanser → ofte lavere effektivitet** ($r = -0,84355$)
- Men: **flere sjanser henger tydelig sammen med flere mål** ($r = 0,92816$)

### 3.3 Sjanser mot (pr. kamp)

- Mot: **Tørum 5,9 / Henriksen 3,8 / Hamrén 4,0 / Eggen 4,4 / Jönsson 5,7**
- Tørum og Jönsson slipper til **nær 6 sjanser pr kamp**.

### 3.4 Sjanse-differanse er nøkkelen

- For å komme over **2 poeng pr kamp** peker analysen på at man bør skape **omtrent dobbelt så mange sjanser som man slipper til**.

## 4. Angrepstype (viser «hvem dere er»)

**Andel sjanser (kontring / etablert / dødball)** varierer tydelig mellom periodene.

Eksempel fra grafene (sjanser pr kamp):

- Kontring: **1,9–2,9**
- Etablert: **2,9–4,3**
- Dødball: **1,3–2,9**

Praktisk tolkning:

- Hamrén/Henriksen har **høy andel etablert** (nesten 50%).
- Eggen-perioden øker kontringssjanser, men risikoen kan slå tilbake i flere kontringer mot.

## 6. Hovedprinsipper (som kan trenes og måles)

1) **Styr etter sjanse-differanse (for–mot), ikke bare «vi skaper mye».**
	- Målet i kampplan bør være «8–10 sjanser for + maks 4 mot» (diff ≈ 4+), fordi materialet peker på at høy diff gir best poengfangst.

2) **Offensiv balanse («def i off») er en angrepsferdighet.**
	- Angrip samlet, ha spillere nær ball, og vær klare til gjenvinning/beredskap ved balltap.

3) **Velg en “stil-kontrakt” før kamp:**
	- **Gjennombruddshissig / kontringsvillig (Eggen):** flere sjanser, men du må tåle flere kontringer mot → krev restforsvar og tydelig 1. forsvarer.
	- **Gjennombruddsavventende / lav risiko (Hamrén):** mer etablert, færre kontringer mot, men ofte «tålmodighetskamper».

4) **Kvalitet > volum i siste tredjedel.**
	- Mer sjansevolum kan gi lavere uttelling, så tren på at sjansene skal være «rene» (store/tydelige) og ikke bare mange.

## 7. Kampdimensjon → KPI (enkelt å føre på benk/etter kamp)

### 7.1 Lag-KPI

- **Sjanser for / mot / diff** (mål: skape ~2x av motstander).
- **Kontringssjanser mot pr kamp** (må ned i «lav-risiko»-kamper).
- **Store sjanser pr kamp** (mål: 2+).

### 7.2 Varsellamper

- «Vi skaper masse, men slipper til mye»: Jönsson 2011 skapte **9,7 sjanser og 2,3 mål**, men slapp også til **5,7 sjanser og 1,5 mål** → bare **1,6 poeng/kamp**.
- Motstander skaper mye **mot etablert forsvar**: i Jönsson-perioden var **45,3%** av sjansene mot etablerte angrep og **2,6 sjanser pr kamp** mot RBKs etablerte forsvar.

## 8. Kampplan (MatchPrep)

1) **Bestem kamp-klima (før dere trener uka)**

- Skal vi være **Eggen-modus** (mer framover/mer folk i angrep) eller **Hamrén-modus** (kontroll + def i off)?

2) **Fast “restforsvar-regel”**

- Når ball er i siste tredjedel: avklar *hvor mange som alltid må være i sikring* (minimum **3+2** er et godt praktisk startpunkt).

3) **Overgangsregler (valg etter brudd)**

- Når vi vinner ball: *har vi ubalanse?* → kontring. Hvis ikke → sikre og etablere (Hamrén-tankegang).

---

## 9. Coaching cues (enkle å rope)

- «**Diff!** Vi skal skape dobbelt så mange som vi slipper til.»
- «**Def i off!** Angrip samlet – vær klare til å vinne igjen.»
- «**Når vi mister: nærmeste trykker, resten sikrer midten.**»
- «**Ikke jag skudd – jag STORE sjanser.**»

---

## 10. Spillestil-modus (3 “brytere” du kan velge per kamp)

1) **Kontroll (Hamrén / play and move):** lav risiko, mer etablert, fokus på tålmodighet + balanse.
2) **Gjennombrudd (Eggen):** mer framover, mer kontringsvilje, flere folk i angrep (krever restforsvar).
3) **Hybrid:** «Hamrén-bunn + Eggen-topp».

---

## 11. Øvelser (rett på feltet)

**UEFA-A11-01: Etablert angrep med restforsvar-regel (10v10/11v11)**

- Regel: alltid X spillere igjen (f.eks. 3+2).
- Poeng: + for skapt stor sjanse, - for kontringssjanse mot (kobler angrep ↔ balanse).

**UEFA-A11-02: Overgangsvalg – «kontring eller sikre?» (7v7+GK)**

- Etter brudd: laget må velge (direkte rute vs sikre + etablere).
- Tell: hvor ofte direktevalg gir sjanse innen 8–10 sek, og hvor ofte dere blir kontra på.

**UEFA-A11-03: «Def i off»-possession med gjenvinning**

- 8v8 (+ jokere): mål bare hvis dere vinner ball tilbake innen 5 sek etter balltap, eller faller i blokk riktig (Hamrén-prinsippet i praksis).
