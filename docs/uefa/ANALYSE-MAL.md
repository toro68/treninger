| Tradisjonelt | Bruk i stedet |
|--------------|---------------|
| 1-er | Keeper |
| 2-er / 3-er | Høyreback / Venstreback (eller generisk: Back) |
| 4-er / 5-er | Stopper |
| 6-er | Sentral midtbane |
| 7-er / 11-er | Høyrevinger / Venstrevinger (eller generisk: Vinger) |
| 8-er | Indreløper / Sentral midtbane |
| 9-er | Spiss |
| 10-er | Offensiv midtbane |

**Eksempler på bruk:**
- «Høyreback overlapper høyrevinger»
- «Sentral midtbane faller mellom stopperne»
- «Spiss faller, offensiv midtbane løper i bakrom»
- «Indreløper støtter vinger i mellomrommet»
- «Sentral midtbane sikrer bak offensiv midtbane»

**Merk:** «Mellomrommet» = sonen mellom motstanderens forsvarslinje og midtbanelinje (på engelsk: «half-space» eller «between the lines»).
