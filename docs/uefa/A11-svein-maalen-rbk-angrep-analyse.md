## Sammendrag (kildetro)

Maalen analyserer målsjanser for og mot RBK i Tippeligaen 2006–2011 for å beskrive hvordan ulike trenerregimer gir ulike angrepsprofiler. Et gjennomgående funn er sammenhengen mellom sjansedifferanse og poengfangst: Det handler ikke bare om å skape mye, men også om å slippe til lite.

## Nøkkeltall (fra råtekst)

| Sjanser mot pr. kamp | Hamrén 4,0 (Jönsson 5,7) |
| Tommelfingerregel (poeng) | Over 2 poeng pr. kamp: «bør man skape dobbelt så mange sjanser som man slipper til» |
| Kontringsandel (RBK-sjanser) | Hamrén 25,5 % → Eggen 30,3 % |
| Store sjanser | Henriksen: 33 % av sjansene var store |

## Praktiske læringspunkter (kildetro formulert)

- Bruk sjanser for/mot (og sjansetype) som felles språk i evaluering – ikke bare mål.
- Velg risikonivå bevisst: angrepstype påvirker både sjanser for og sjanser mot.
- Koble tall til video: Hvilke mønstre gir sjanser for, og hvilke situasjoner gir sjanser mot?
