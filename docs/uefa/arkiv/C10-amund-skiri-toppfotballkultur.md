# Toppfotballkultur i praksis – Amund Skiri (UEFA A 2015)

Kilde: «Hvordan skape en toppfotballkultur? En studie av lagbygging og lederskap i Aalesund 2001–2002.» Notatet trekker ut kamprelevante elementer: hva som må inn i MatchPrep, CoachingTips, TeamOrganization og Roles for å forankre kultur, lederskap og kampforberedelse.

## 1. Kulturpilarer (kan gjenbrukes som kampmantra)
- **Klart «hvorfor»**: Aafk definerte opprykk som eksplisitt mål og repeterte det i alle møter. Kampseksjonen bør starte med «hvorfor denne kampen er kritisk» for å koble spillerne på.
- **Semiproffmodell med ansvar**: Deltids- og heltidsgrupper måtte rapportere egen trening. Bruk i MatchPrep: marker hvilke spillere som må gjøre ekstra innsats før kamp pga. jobb/skole.
- **Høye krav + sosial trygghet**: Normark ble beskrevet som «sær, tydelig, morsom» – kombinerte tøffe krav med humor. CoachingTips bør minne om tone: «hard på standard, myk på person».

## 2. Lagbygging → kampdrift
- **Gruppe → lag**: 2001-stallen ble bevisst satt sammen av ulike typer (lokale kulturbærere + rutinerte tilflyttere). TeamOrganization kan ha felt «hvem er kulturanker i start XI?».
- **Ritualer**: Felles frokost, kortspill, interne konkurranser, «samlingsrop» før kamp. Legg inn sjekkliste for kampdag-ritualer for å sikre samme energi.
- **Motgangsprosedyre**: Når laget fikk en dårlig uke gikk trenerteamet tungt inn med tydelig plan for neste kamp (spesialmøter, video). Kampseksjonen bør ha en modul «reaksjon etter tap» med konkrete oppgaver.

## 3. Lederskap direkte inn i kampseksjonen
- **Rolleavklarte ledere**: Kaptein (Skiri) håndterte garderobe + banefokus, mens assistent (Stig Monsen) var bindeledd. Lag kampplan med navngitte ansvar for kommunikasjon mot dommer, mot benk, mot unge spillere.
- **Spillerinvolvering**: Laget brukte «spillerforum» før kamp for å diskutere taktikk og fordeling av ansvar. MatchPrep kan kreve at ledergruppen møtes kvelden før for å sikre eierskap.
- **Feedback-loop**: Etter kamp ble sannheten adressert direkte (ingen «white lies»). PostMatch bør kopiere dette: ett punkt «sannhet først» før man går til video.

## 4. Kampplan fra Aafk 2002 (kan adapteres)
- **Press**: laget var ekstremt aggressivt hjemme (Kråmyra) men kontrollerte tempo borte. Banner i TeamOrganization: «Hjemme = aggressivt høyt press 20 min, Borte = kontrollert blokk + hurtige overganger».
- **Spillertyper**: Normark ønsket «vinnertyper, brede skuldre, litt galskap». Roles kan få et ekstra felt «mental trigger» per posisjon (eks: «stopper: vinn alle dueller første 15 min»).
- **Rotasjon**: trenerne var åpne om at enkelte kamper krevde andre personligheter (fysisk vs teknisk). MatchPrep kan ha «typevalg» felt der vi begrunner hvorfor spiller X starter pga. kampens karakter.

## 5. Kampforberedelser
- **Semipro-logistikk**: Morgentrening/ettermiddag, spillerne måtte planlegge hvile. Kampseksjonen bør inkludere hvileskjema før kamp (særlig for de som jobber/studerer).
- **Video + referanser**: selv med begrenset ressurser ble motstander analysert (telefon til andre trenere, hand-cut VHS). I MatchPrep, legg inn sjekk «er motstanderinfo verifisert?».
- **Cup-erfaring**: 7–6-kampen mot Brann ga læring: «aldri slutt før slutt». CoachingTips kan bruke sitatet som mental påminnelse i sluttfasen.

## 6. Konkret sjekkliste å bake inn
1. **Kulturanker** identifisert i start XI?
2. **Semipro-tilpasning** gjort (hvile, reise, jobb)?
3. **Kampdag-ritualer** planlagt (felles måltid, oppmøte, rop)?
4. **Lederskap**: hvem tar dommerdialog, hvem snakker med unge, hvem holder fokus i pausen?
5. **Motgangsplan**: hva gjør vi hvis vi slipper inn først?
6. **Bra-pluss**: lagets «galskap» element – et grep som viser at vi tør (lang innkast, corner-variant, mental trigger).

## 7. Sitater/tekstsnutter klare for kampseksjonen
- «Vi ble slitne, men lagbyggingen holdt oss sammen.»
- «Semiproffmodellen tvang oss til å være proffe i hodet.»
- «Kulturen var at du sier ting rett ut, men alltid med respekt.»

Bruk sitatene i CoachingTips- eller Roles-komponentene som korte påminnelser om standardene som bygde Aafks toppfotballkultur.
