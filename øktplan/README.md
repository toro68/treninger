# Instruksjoner for øktplaner

Her finner du maler og retningslinjer for å lage øktplaner.

## Filnavn

Bruk følgende format for filnavn:

`YYYY-MM-DD-tema.txt`

For eksempel: `2026-02-03-hurtig-angrep.txt`

## Standard øktstruktur

En standard økt varer i 90 minutter og følger denne strukturen:

1.  **Skadefri 5 min) og styrke med Tor Harald (5 min)**: Forebyggende øvelser.
2.  **Rondo/smålagspill (10 min)**: 
- Hvis antallet på trening kan deles på 3, så går det opp. (Joker eller skjeve lag (f.eks. 3v3v4)for å få det til å gå opp)
- Fokus på: 
3.  **Stasjoner med fokusområder (20-30 min)**: 2-3 stasjoner med øvelser som fokuserer på dagens tema.
4.  **Spill (30-40 min)**: Spill på mindre flater eller full bane, med regler som fremmer dagens tema.
5.  **Avrunding /tøying (5 min)**: Rolig nedjogg og uttøyning.
