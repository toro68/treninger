# Fysisk dimensjon – Jørgen Ingebrigtsen (UEFA A 2012)

Kilde: «Den fysiske dimensjonen i fotball – oversiktsstudie». Notatet oversetter funn om høy-intensitet, energiomsetning, testing og trening til kampseksjonen.

## 1. Høy-intensitet og posisjon
- Topplag utfører mer HI-løp; krav varierer per posisjon. TeamOrganization: planlegg bytter og belastning per posisjon (f.eks. kant = mest sprint). MatchPrep: referer til posisjonelle HI-krav.
- CoachingTips: minne spillere om energistyring (eks: når å sprinte vs spare). 

## 2. Energisystemer
- Aerob + anaerob viktige. Kampseksjonen kan justere kamplan (press vs lav blokk) basert på uke-belastning. MatchPrep: noter «energi-fokus» (tung uke → kontroller tempo). 

## 3. Testing/Kapasitetsprofil
- Bruk testdata (VO2, 40m, hopp) til å styre kamproller. TeamOrganization: markér spillerens status (grønn/gul/rød) basert på test/velvære.

## 4. Oppvarming/re-warm
- Ingebrigtsen refererer til Mohr (2004) om halvtime oppvarming. Kampseksjon: plan for re-warm på benken (sykkel, mini-spill). 

## 5. Repeated sprint & small-sided
- HI-løp trent via small sided + repeated sprint. Kampplan: hold benk klare (varme) for sprints i sluttfase.

## 6. Implementering
- MatchPrep: «Fysisk modul» (HI-krav, bytteplan, rewarm). 
- Roles: legg inn individuelle kapasitetstiltak (spiller X må begrense antall sprints). 
- CoachingTips: fortell spillerne om energihåndtering, hydrering, rewarm. 
- PostMatch: logg HI-data (GPS) vs plan, bruk til planlegging.

Dette notatet gjør Ingebrigtsens forskningsoversikt anvendbar for kampplanen.
